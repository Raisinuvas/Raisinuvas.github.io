前置知识2 Java跨平台原理

编程——编译（将.java转化为机器语言）——运行

Java是==混合型==编译运行，Java并非直接运行在系统中，而是运行在自带的虚拟机中，针对不同的系统安装不同的虚拟机即可。

因此，程序的运行需要JVM(Java虚拟机)、核心类库、开发工具，这三部分整体称为JDK。

3. 标识符

一些软性建议：

  • 小驼峰命名——适用于方法和变量：标识符是一个单词的时候，全部小写；标识符由多个单词组成的时候，第一个单词首字母小写，其他单词首字母大写。
  • 大驼峰命名——适用于类名：标识符是一个单词时，首字母大写；标识符由多个单词组成，每个单词首字母大写。
